# Visual System

Metro uses a dark control-room background, subtle grid, luminous curved tracks, circular stations, restrained glow, and smooth train silhouettes.

Retro uses a pixel-grid background, block trains, square stations, small animated environmental lights, and monospace interface styling.

Tokens live in `src/styles/global.css` for spacing, borders, panels, focus states, and layout sizing. The map dominates desktop layouts, while narrow viewports collapse secondary panels.

Animations are calm, loop-safe, and disabled when reduced motion is requested.
