# V1 Acceptance Criteria

- The user can launch demo mode.
- The user can enter `owner/repository`.
- The user can paste a GitHub repository URL.
- Invalid repository input shows a clear error.
- Metro and Retro themes are available and visually distinct.
- The map shows lines, stations, moving trains, activity feed, and train details.
- Pause, speed, fullscreen, and reduced-motion behavior exist.
- Public GitHub data is fetched directly from the browser.
- No API key, account, telemetry, backend, or database exists.
- `pnpm lint`, `pnpm typecheck`, `pnpm test`, and `pnpm build` pass before completion.
- Public commit activity appears on the transit map after GitHub publishes it through the public API.
