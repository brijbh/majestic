# Product Brief

Git Transit is a visual repository activity experience for public GitHub repositories. It is designed for portfolio use and ambient large displays, not for Git operations or project management.

Audience: developers, maintainers, demo viewers, and teams who want a glanceable sense of repository movement.

Journey: enter a public repository, launch the map, watch workstreams move, inspect trains, switch themes, and use demo mode when GitHub data is unavailable.

Included: public GitHub data, demo mode, Metro theme, Retro theme, activity feed, train inspector, pause, speed, fullscreen, and reduced-motion support.

Excluded: private repositories, authentication, analytics, telemetry, hosted sharing, Git operations, issue editing, productivity scoring, and AI summaries.
