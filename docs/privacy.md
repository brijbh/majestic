# Privacy

Git Transit has no custom backend, no user accounts, no hosted database, no telemetry, no analytics, no tracking pixels, and no advertising.

The browser requests public repository information directly from GitHub. GitHub receives those requests as part of normal API access.

Local storage is used only for non-sensitive preferences: last repository, selected theme, animation speed, and reduced-motion preference. Complete repository API responses are not permanently stored by Majestic.

Users may optionally save a GitHub personal access token in local storage to raise GitHub API limits or access repositories they own. Treat this token like a password. It is sent directly from the browser to GitHub and is not sent to Majestic infrastructure.
